## 사용된 기능
* :clock4: 실시간 Clock 기능
* :recycle: LocalStorage를 활용한 DB 데이터 출력 및 저장
* :link: Weather-API를 호출하여 자신이 위치한 지역과 날씨 정보 불러오기
* 새로고침을 시도할 때마다 백그라운드 이미지 랜덤으로 변경
* 아침, 점심, 저녁 순으로 출력 문구 변경
* :part_alternation_mark: Todo-List CRUD 기능
